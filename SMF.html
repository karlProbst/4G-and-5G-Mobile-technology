SMF

Category: 5G / core
Executive Summary:
The Session Management Function (SMF) orchestrates data sessions and IP address allocation in the 5G core. It acts as the logical bridge instructing the data plane how to behave.

Advanced Technical Definition

The SMF receives policy directives from the PCF and mobility context from the AMF to select and control the UPF. It dictates the exact routing behavior of the massive data streams traversing the gNB and the Next Generation Core (NGC). By logically partitioning sessions, the SMF is a mandatory component for instantiating customized, end-to-end Network Slicing.
