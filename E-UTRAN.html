E-UTRAN

Category: 4G / architecture
Executive Summary:
E-UTRAN is the radio access network for 4G LTE. It handles all physical radio communications between the mobile device and the core network.

Advanced Technical Definition

E-UTRAN operates as the RF access component of the Evolved Packet System (EPS), comprising a synchronized mesh of eNodeB (eNB) base stations. It achieves up to 30 bps/Hz spectral efficiency by utilizing OFDMA for downlink and SC-FDMA for uplink. The E-UTRAN completely lacks centralized radio network controllers, dynamically routing traffic straight to the Evolved Packet Core (EPC) while relying on X2 interfaces between eNBs to execute 0ms handover interruptions.
