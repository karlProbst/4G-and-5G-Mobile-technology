UPF

Category: 5G / core
Executive Summary:
The User Plane Function (UPF) is the primary data routing and forwarding engine in the 5G network. It handles the heavy lifting of raw internet traffic without analyzing control signals.

Advanced Technical Definition

The UPF serves as the high-throughput anchor point for Intra-RAT mobility and deep packet inspection. Instructed entirely by the SMF via SDN methodologies, it routes multi-gigabit eMBB traffic directly from the gNB to external Data Networks. Because it is decoupled from the control plane (AMF/SMF), the UPF can be geographically distributed to the network edge to satisfy strict URLLC latency bounds.
