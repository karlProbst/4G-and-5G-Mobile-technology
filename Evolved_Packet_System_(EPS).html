Evolved Packet System (EPS)

Category: 4G / architecture
Executive Summary:
The EPS is the complete end-to-end, all-IP 4G network system. It comprises both the radio access network and the core network to provide flat, rapid data routing.

Advanced Technical Definition

The EPS unifies the E-UTRAN (radio access) and the Evolved Packet Core (EPC) to provide persistent, low-latency (<5ms) IP connectivity. It officially eliminates the legacy circuit-switched domains of 3G, replacing them entirely with packet-switched pathways. Traffic flows securely from the User Equipment (UE), through the eNodeB (eNB) arrays, and is anchored by EPC signaling nodes like the MME and PCRF.
