NSA

Category: 5G / core
Executive Summary:
Non-Standalone (NSA) is a transitional 5G deployment model intended for rapid commercial rollout. It pairs new 5G radio antennas anchored to the existing 4G core infrastructure.

Advanced Technical Definition

Specified under 3GPP Option 3x families, NSA structurally links the 5G-NR.html">5G-NR gNB and the legacy 4G eNodeB (eNB) back to the Evolved Packet Core (EPC). This allows operators to provide gigabit eMBB speeds immediately without building the Next Generation Core (NGC). In this topology, the LTE eNB typically acts as the master node handling MME control signaling, while the gNB aggregates heavy data planes directly or via the EPC.
