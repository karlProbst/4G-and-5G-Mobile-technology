Evolved Packet Core (EPC)

Category: 4G / architecture
Executive Summary:
The EPC is the core network framework for 4G LTE. It provides packet routing, mobility management, and strict QoS enforcement.

Advanced Technical Definition

Acting as the central intelligence of the Evolved Packet System (EPS), the EPC manages mobile sessions and prevents packet delay. It interacts continuously with the E-UTRAN via S1 interfaces, controlling the eNodeB (eNB) array. Core responsibilities are logically distributed among the MME (control-plane signaling), HSS (cryptographic subscriber database), and PCRF (policy enforcement). The EPC is also heavily utilized as the traffic anchor in 5G NSA deployments.
