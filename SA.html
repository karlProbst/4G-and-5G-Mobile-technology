SA

Category: 5G / core
Executive Summary:
Standalone (SA) is the ultimate, fully independent 5G network deployment model. It couples the 5G-NR radio directly to a cloud-native, software-defined 5G core.

Advanced Technical Definition

Bypassing the hardware limitations of the Evolved Packet Core (EPC) entirely, SA connects the gNB directly to the Next Generation Core (NGC). It natively leverages the AMF, SMF, and UPF nodes. This clean-slate, virtualized architecture is absolutely mandatory for unlocking true 5G capabilities, including dynamic end-to-end Network Slicing, sub-millisecond URLLC performance, and mMTC density.
