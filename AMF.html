AMF

Category: 5G / core
Executive Summary:
The Access and Mobility Management Function (AMF) handles all control-plane signaling in the 5G core. It acts as the software gatekeeper for user authentication, registration, and mobility.

Advanced Technical Definition

Serving as the cloud-native 5G evolution of the 4G MME, the AMF manages UE registration, roaming state, and authentication via the AUSF/UDM. It authorizes devices connecting via the gNB and communicates extensively with the SMF to establish distinct session routing paths. As a Virtualized Network Function (VNF), it enables the flat, distributed Network Architecture required by 5G.
