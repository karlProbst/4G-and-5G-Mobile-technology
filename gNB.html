gNB

Category: 5G / radio
Executive Summary:
The next-generation NodeB (gNB) is the primary 5G base station hardware. It broadcasts the highly capable 5G-NR signal and connects directly to the 5G core network.

Advanced Technical Definition

Replacing the legacy 4G eNodeB (eNB), the gNB coordinates radio resources using massive MIMO and mmWave beamforming (e.g., 28 GHz). It physically separates control and user planes, routing NAS signaling directly to the AMF and user plane multimedia payloads straight to the UPF. The gNB adapts seamlessly to dynamic Network Slicing profiles for eMBB, URLLC, and mMTC traffic.
