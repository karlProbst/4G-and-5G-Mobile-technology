Link: [https://karlprobst.github.io/4G-and-5G-Mobile-technology/](https://www.google.com/search?q=https://karlprobst.github.io/4G-and-5G-Mobile-technology/)

* 4G Architecture: EPS, EPC, E-UTRAN, eNodeB, MME, HSS, PCRF.

* 5G Core: AMF, SMF, UPF, Network Slicing, NSA, SA.
* 5G Radio: 5G-NR, gNB.
